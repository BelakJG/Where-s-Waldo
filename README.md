# Where's Waldo

[Project Link](https://www.theodinproject.com/lessons/react-new-where-s-waldo-a-photo-tagging-app)

This project is meant to practice tying a react front end with a backend database (Rails in this project).

It will be a game of where's waldo or similar style games where you will have a find a character,
and it will let you save your score into a database to be saved and viewable by anyone.
